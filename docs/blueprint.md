# **App Name**: ReactMail

## Core Features:

- User Authentication: Secure user login and registration using email and password.
- Inbox Display: Display a list of received emails with sender, subject, and a brief preview.
- Email Composition: Allow users to compose and send new emails with rich text editing capabilities.
- Email Sending: Send an email and storing this to the selected Cloud SQL database.
- Email Receiving: Receiving an email to the respective mailbox.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and trust.
- Background color: Very light indigo (#E8EAF6), almost white, to avoid distractions from content.
- Accent color: Muted blue-green (#4CAF50) for important action items and highlights.
- Body text: 'Inter' sans-serif for clean, readable body text.
- Headline text: 'Space Grotesk' sans-serif for a slightly modern, machined headline aesthetic; use 'Inter' for body text when longer lines of text are involved.
- Use simple, minimalist icons for actions like send, reply, delete, and compose.
- Implement a clean, well-organized layout with clear separation between inbox, compose, and other sections.
- Subtle transitions and animations for actions like sending and receiving emails, providing smooth user experience.
